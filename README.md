# 🎮 Troll Game (TUJ Game Jam 2025)

## 🏆 Achievement

Our team achieved **1st Place** at the TUJ Game Jam with this subversive boss-battle game, created for the theme **"Life is a lie."**

Instead of relying on traditional gameplay mechanics, we engineered a **creative, empathy-based win condition**. This unique approach highlighted our ability to **innovate on a tight deadline** and deliver a game that surprised both players and judges.

---

## 🚀 Tech Stack

- **Language:** Python 3.13
- **Library:** Pygame 2.6.1

---

## 🕹️ How to Play

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/troll-game.git
   cd troll-game
   ```
2. Install dependencies:
   pip install pygame
3. Run the game:
   python main-game-file.py
